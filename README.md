# 🍰 GUNGO'S

Una tienda online desarrollada como proyecto de práctica, enfocada en ofrecer una experiencia sencilla e intuitiva para visualizar productos y realizar compras mediante un carrito.

---

## 📋 Descripción

GUNGO'S es una aplicación web que permite a los usuarios explorar un catálogo de productos, filtrarlos por categorías, agregarlos al carrito y gestionar su compra de manera simple.

Los datos de los productos se almacenan en **Firebase Firestore**, mientras que las imágenes pueden obtenerse desde **Firebase Storage**. El carrito utiliza **LocalStorage**, permitiendo conservar los productos incluso al recargar la página.

---

## ✨ Funcionalidades

- 📦 Catálogo de productos.
- 🔍 Filtrado por categorías.
- 🛒 Carrito de compras.
- ➕ Agregar productos al carrito.
- ➖ Modificar cantidades.
- 🗑 Eliminar productos.
- 💾 Persistencia del carrito mediante LocalStorage.
- ☁ Base de datos en Firebase Firestore.
- 🖼 Imágenes almacenadas en Firebase Storage.
- 📱 Diseño responsive.

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase
  - Firestore
  - Firebase Storage
